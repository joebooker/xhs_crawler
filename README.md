# xhs_crawler
小红书爬虫，小红书数据挖掘，数据分析

# 简介
本项目主要针对小红书web端进行数据采集，包括但不限于笔记搜索，笔记详情，笔记评论，笔记子评论，用户主页等。支持保存到Excel，csv等文件类型，支持下载图片视频等。
